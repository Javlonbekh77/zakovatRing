# **App Name**: Zakovat Ring

## Core Features:

- Admin Panel: An admin interface to add questions, answers (hidden in a grid), and letter-reveal questions to a Firestore database.
- Game Lobby: Teams enter a code to join a specific game instance, displaying team names.
- Question Display: Displays the main question to both teams along with a masked answer grid.
- Team Answer Submission: Teams submit answers, validated against the hidden answer, earning points from a central pool, which decreases over time.
- Letter Reveal: A letter is revealed either at regular intervals or when teams correctly answer a 'letter-reveal question'.
- Real-time Scoreboard: A scoreboard updates in real-time showing team scores.

## Style Guidelines:

- Primary color: Dark blue (#3F51B5), for a trustworthy, intellectual feeling.
- Background color: Very light blue (#E8EAF6).
- Accent color: A slightly desaturated violet (#7E57C2) will indicate active interactive elements.
- Body and headline font: 'PT Sans', sans-serif, combines modernity with approachability.
- Simple, clear icons to represent different question categories.
- Clean, grid-based layout for clear information presentation.
- Subtle animations to highlight correct answers and score updates.
